Milkodeについて

コードを読むことの重要性は年々向上

しかし、読むためのツールは変わっていない・・

  • grepは時間がかかる
  • Webのソースコード検索エンジンはAND検索が出来ない、プライベートなコードに使えない
  • "grep" + "Web検索" = ??

もっとコードリーディングを簡単にしたい!!

Milkodeの特徴

  • ソースコードを読むための検索エンジン
  • 高速 (20,000ファイル位であれば1秒以内に検索)
  • 行指向 (AND検索)
  • エディタでもブラウザでも

ソフトウェアの構成

  • Ruby, Haml, JavaScript
  • rroonga (ストレージ & 検索エンジン)
  • Thor (コマンドライン)
  • Sinatra (Webアプリ)
  • CodeRay (ソースコードの色付け)
  • 他にもたくさんのgemのおかげで出来てます

rroongaの使い方

何故rroongaを使ったの？

  • groongaがカラムストア機能付きだから！！
  • 別途データベースのインストールが不要
  • 'gem install rroonga' でgroongaもまとめてインストール出来る

ローカルアプリとも相性がよい

実装の解説

スキーマ

  • そんなに変わった事はしてません
  • 1ファイル == 1レコード としてテーブルに登録
  • 拡張子はソースコード検索にとって重要なので専用カラムを作成

どうやって一行を絞り込むの?

  • rroongaで可能性のあるレコードを絞り込み
  • 行の絞り込みはRuby上で行う
  • これで目的の一行を見つけ出すことが出来ます

Tips

  • トークナイザーはTokenBigramSplitSymbolAlphaDigitを使いましょう
  • コード検索の場合、'bigrobo'に'robo'でマッチ出来た方がよい
