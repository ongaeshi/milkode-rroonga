Milkodeについて

コードを読むことの重要性は年々向上


しかし、支援するためのツールは変わっていない・・

  • grepは時間がかかる
  • Webのソースコード検索エンジンはAND検索が出来ない、プライベートなコードに使えない
  • "grep" + "Web検索" = ??

もっとコードリーディングを

簡単にしたい!!

Milkodeの特徴

  • ソースコード検索エンジン
  • 高速 (20,000ファイル位であれば1秒以内に検索)
  • 行指向 (AND検索)
  • エディタでもブラウザでも

ソフトウェアの構成

  • Ruby
  • rroonga (ストレージ & 検索エンジン)
  • Thor (コマンドライン)
  • Sinatra (Webアプリ)
  • CodeRay (ソースコードの色付け)
  • 他にもたくさんのgemのおかげで出来てます

デモ

rroongaの使い方

何故rroongaを使ったの？

  • groongaがカラムストア機能付きだから！！
  • 別途データベースのインストールが不要
  • 'gem install rroonga' でgroongaもまとめてインストール

ローカルアプリと相性がよい

何故rroongaを使ったの？(2)

  • Rubyだけで書ける
  • Rubyらしく読み書きしやすいAPI
  • ブロックで検索式を記述

Cは仕事以外で使いたくない・・

実装の解説

スキーマ

  • そんなに変わった事はしてません
  • 1ファイル == 1レコード としてテーブルに登録
  • 拡張子はソースコード検索にとって重要なので専用カラムを作成

どうやって一行を絞り込むの?

  • rroongaで可能性のあるレコードを絞り込み
  • 行の絞り込みはRuby上で行う
  • これで目的の一行を見つけ出すことが出来ます

Tips

  • トークナイザーは'TokenBigramSplitSymbolAlphaDigit'を使いましょう
  • 'BigRobot'に'robot'でマッチ出来ません

インストール

$ gem install milkode
